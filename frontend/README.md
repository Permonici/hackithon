# Frontend

React + Tailwind konzole pro demonstraci XDENT asistenta.

## Obrazovky a funkce

- chat pro dotazy zakazniku,
- lokalni chat history,
- vlozitelny chat widget v pravem dolnim rohu,
- sila zdroje primo u odpovedi v chatu,
- lokalni ulozeni dat pacienta v prohlizeci,
- light/dark mode a zvetseni pisma,
- hlasovy vstup pres mikrofon pouze v cestine,
- volitelne ceske cteni odpovedi nahlas,
- rozkliknuti zdroju primo u odpovedi,
- realtime agent timeline,
- strict mode,
- eskalacni balicek pro 2. uroven podpory,
- status OpenAI/Qdrant.

## Lokalni spusteni bez Dockeru

```powershell
cd frontend
npm install
npm run dev
```
